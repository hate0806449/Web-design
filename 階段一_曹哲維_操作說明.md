## 3. 啟動方式（驗證階段一能跑）

```powershell

# 裝 Python 依賴
pip install -r requirements.txt

# 啟動 FastAPI（會在 8000 port）
python -m uvicorn main:app --reload --port 8000
```

```powershell

# Next.js 預設跑在 http://localhost:3000
```

## 5. 可以驗證的 API

```powershell
# 健康檢查

# → {"status":"ok"}
```
